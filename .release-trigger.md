# Trigger 1.35.4 release

## Changes

- Implemented manual refresh and 30s configurable polling for Sessions and App Instances management pages.
- Added real-time K8s status and message columns to management tables.
- Added session ID column and workbench member list (owner) to the Sessions table.
- Optimized performance by utilizing batch user fetching in the Workbench table.
- Improved UI consistency for status badges and table headers across administration views.
- Fixed TypeScript type guards and resolved various linting issues.

## Type

Patch release - Feature enhancements and performance optimizations.
